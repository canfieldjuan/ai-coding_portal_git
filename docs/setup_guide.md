# Setup Guide

## Prerequisites
- Python 3.8+
- OpenRouter API key

## Installation
1. Run setup script: `python setup.py`
2. Configure environment: Edit `.env` file
3. Start server: `python run.py`
4. Test endpoints: `python test_api.py`

## Configuration
All configuration is done via environment variables in `.env` file.
